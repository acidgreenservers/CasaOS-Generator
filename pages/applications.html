Your exported ZIP contains every saved application organized into its own directory. Each directory includes:

  • docker-compose.yml — The full CasaOS app configuration
  • icon.png — Application icon (if available)
  • screenshots/ — Application screenshots (if available)

Ready to use as a CasaOS app store. Upload the extracted contents to a GitHub repository, configure your CasaOS to point to it, and your applications will appear as installable apps — no additional transformation needed.
